# Global Population Trend Analysis Using Excel

## Project Overview

This project analyzes historical population data across multiple countries to identify demographic patterns, population stability, and long-term growth trends. Using Microsoft Excel, statistical measures were calculated and interpreted to classify each country's population behavior over time.

The project demonstrates practical data analysis skills, descriptive statistics, trend interpretation, and business reporting techniques commonly used in Data Analyst and Business Analyst roles.

---

## Business Objective

Organizations and policymakers often need to understand how populations change over time to support strategic planning, economic forecasting, and resource allocation.

The objective of this project is to:

* Summarize population characteristics for each country.
* Identify population stability and fluctuations.
* Detect increasing or decreasing population trends.
* Create a structured reporting table for decision-making.

---

## Dataset

The dataset contains population figures for multiple countries across several years.

Each country's historical population data was analyzed to generate statistical summaries and trend classifications.

---

## Analysis Performed

### Descriptive Statistics

For every country, the following metrics were calculated:

* Mean (Typical Population Size)
* Median (Central Population Value)
* Mode (Most Common Population Size)
* Minimum Population
* Maximum Population

### Data Quality Handling

* Managed missing values and calculation errors.
* Assigned "None" where no mode existed.
* Assigned "NA" when statistical measures could not be calculated.

### Population Trend Classification

Countries were categorized using business rules:

#### Extreme Population Fluctuation

Applied when:

* Maximum Population > 1.5 × Average Population

OR

* Minimum Population < 0.5 × Average Population

#### Increasing Population Trend

Applied when:

* Median Population > Average Population

#### Decreasing Population Trend

Applied when:

* Median Population < Average Population

#### No Trend

Applied when:

* No conditions were satisfied
* Calculation errors occurred
